# シフト自動作成 Web アプリ（Executive Lounge）

ホテル Executive Lounge の月間シフト表を **ブラウザだけ** で作成・編集・Excel入出力できる、
ビルド不要の静的サイトです。サーバー不要、`index.html` を開くだけで動きます。

公開URL（GitHub Pages, main/root）:
**https://bokecatland-ctrl.github.io/shift/**

---

## 特長

- **ビルド不要**：素の HTML / CSS / JavaScript。リポジトリ・ルート直下に配置。
- **色付き Excel の読み書き**：[ExcelJS](https://github.com/exceljs/exceljs)（MIT）を `vendor/` にベンダリング（CDN 不要・オフライン可）。
- **個人情報をコミットしない**：氏名は **この端末の `localStorage` と手元の `.xlsx` のみ** に保存。
  リポジトリには一切含めません（`.gitignore` で `*.xlsx` を除外）。

## 機能

1. **スタッフ / 役職 / 希望休の管理**（`localStorage` 永続化）
   - 役職：`JP4 > E2 > E1`（社員）、`PT`（アルバイト）。
   - `NG G3/G4` は「今月だけの他部署」なので自動生成では無視（取込時は表示のみ）。
   - 希望休はカレンダーをクリックして指定。
2. **下書きの自動生成**
   - **JP4** → `75/1000`（アシマネ枠）を毎日 1 人ローテーション。
   - **社員(E2/E1)** → **2コマセット**（同一人物が「N日 13:00 → 翌 5:30」）を毎日 2 件。
     これで毎日 `75/1300`(遅番)×2 ＋ `75/0530`(早番)×2 になります。
   - 希望休を尊重し、勤務でない日は `/`（公休）で埋める。
   - **PT（アルバイト）は自動生成の対象外**（希望ベースで手動入力）。
   - **生成ルール（ベース設定・localStorage 保存）**
     - **今月の公休数**：1人あたりの目標公休数を設定（月により変更可。既定 10）。
     - **余りコマを土曜優先**：2コマ需要だけでは勤務が目標に届かない分を「余りコマ」
       （中番・既定 `75/1200`）として空き日に追加。配置先を**土曜優先**にできる。
   - **連勤ルール**：なるべく **5連勤** まで。やむを得ず **6連勤** になる場合は
     画面でアラート（橙下線＋バナー）。**7連勤は生成で絶対に作りません**
     （手入力で7連勤になった場合は赤表示で強く警告）。
3. **手直し＋手入力固定＋警告＋下部集計**
   - セルをクリックしてシフトを変更（色付きパレット／自由入力）。
   - **手入力したセルは「固定（🔒）」**され、**自動生成で上書きされません**。
     固定セルは所与の条件として日別の必要数に数え、不足分だけを自動で埋めます
     （例：1日にある人の早番5:30を入れて生成すると、それは動かさず周りを組む）。
     固定した遅番には 2コマセットの相方（翌5:30）が自動補完されます。
     固定セルは濃い縁＋右上マークで表示。ポップオーバーから固定解除も可能。
   - 1日の人数が必要数（遅番2・早番2・アシマネ1）と違うと**色で警告**。
   - 画面下部に**集計**（各シフトの日別人数・社員出勤計）を再現。
4. **Excel 入出力**
   - 既存 `.xlsx` を取り込み（日付ヘッダ行を自動検出）。
   - **同じ見た目**（色・列幅・COUNTIF 集計）で `.xlsx` を書き出し。

## シフトコードと色

| コード | 意味 | 色 |
|---|---|---|
| `75/1300` | 遅番 13:00→ | 🟦 `#33CCFF` |
| `75/0530` | 早番 →翌5:30 | 🩷 `#FF99FF` |
| `75/1000` | アシマネ 10:00（JP4） | 🟧 `#FFC000` |
| `/` | 公休 | 🟩 `#00FFCC` |
| `BD` | 誕生日休 | 🟥 `#FF0000` |

その他のコード（`75/0800` 等）は自由入力でき、灰色で表示・そのまま Excel に保存されます。

## 使い方

1. このリポジトリを開く（Pages 公開後は上記 URL）。ローカルなら `index.html` をダブルクリック。
2. 「役職テンプレで初期化」または手入力でスタッフを登録（氏名は端末内のみ保存）。
3. 必要なら希望休を指定 → 「下書き自動生成」。
4. セルをクリックして手直し。下部集計と警告を見ながら調整。
5. 「.xlsx 書出」で配布用ファイルを保存（手元のみ。リポジトリには入りません）。
6. 既存表があれば「.xlsx 取込」で読み込み、同じ見た目で再出力できます。

## GitHub Pages の有効化

このアプリは `main` ブランチのルートで動きます。Pages 公開手順：

1. リポジトリ **Settings → Pages**
2. **Source: Deploy from a branch**
3. **Branch: `main` / `(root)`** を選択して Save
4. 数分後 `https://bokecatland-ctrl.github.io/shift/` で公開

> 現在の開発は `claude/shift-auto-generation-app-3f2c68` ブランチで行っています。
> `main` にマージしてから上記設定を行ってください。

## ファイル構成

```
index.html            画面
css/style.css         スタイル
js/config.js          シフト定義・色・日付ヘルパー
js/store.js           localStorage 永続化
js/generator.js       下書き自動生成アルゴリズム
js/grid.js            表描画・セル編集・警告・集計
js/xlsx.js            .xlsx 取込/書出（ExcelJS）
js/app.js             UI 配線
vendor/exceljs.min.js ExcelJS 本体（MIT）
vendor/exceljs.LICENSE
```
